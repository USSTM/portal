# Give Event control to one Owning Club

Each Event has one Owning Club whose authorized Members control its details, additional organizer list, and lifecycle. Other Organizing Clubs receive attribution and visibility but no modification authority. Only Administrators may transfer ownership to another active Club, and the transfer is audited. This preserves the legacy system's single-creator control without tying authority to an individual or introducing ambiguous collaborative ownership.

The portal will not keep a global active-Club context. Members see Events across all Clubs they can access and choose an Owning Club explicitly when creating an Event, so authorization always follows the record rather than hidden session state.
