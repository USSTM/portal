## Agent skills

### Issue tracker

Issues and specs for this repo live as markdown files under `.scratch/`. See `docs/agents/issue-tracker.md`.

### Triage labels

The canonical triage labels are `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, and `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context domain docs live at the repo root in `CONTEXT.md`, with ADRs in `docs/adr/`. See `docs/agents/domain.md`.

### Commits

Do not stage/commit changes, leave that to the human.
